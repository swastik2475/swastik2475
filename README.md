### 🔬 LLM Interpretability Toolkit

Built an open-source toolkit to visualize transformer attention across **12 layers × 12 heads** in GPT-style models. Features attention heatmaps, gradient-based token attribution using PyTorch + HuggingFace Transformers, and an interactive Streamlit dashboard with automated CI/CD via GitHub Actions.

`PyTorch` `HuggingFace` `Streamlit` `GitHub Actions`

[→ View Repo](https://github.com/swastik2475/llm-interpretability-toolkit)

### 🐀 RodentAI — Vehicle Rodent Detection System
> *Custom YOLOv11n · 77.4% mAP@50 · 79% Precision · 72% Recall*

Real-time rodent detection around parked vehicles using a custom-trained **YOLOv11n** model (~400 image dataset). Works in low-light and cluttered environments with optional ultrasonic deterrence trigger. Includes confusion matrix, training curves, and reproducible validation pipeline.

`YOLOv11n` `Ultralytics` `OpenCV` `Jupyter`

[→ View Repo](https://github.com/swastik2475/rodent-ai)

`FAISS` `Sentence Transformers` `FastAPI` `Streamlit` `LLM`
